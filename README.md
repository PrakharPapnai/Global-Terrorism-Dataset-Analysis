# Global Terrorism Dataset Analysis

## Overview

This project analyzes the Global Terrorism Database (GTD), a comprehensive collection of terrorist incidents worldwide from 1970 to 2017. Using Python and Power BI, the project explores temporal and geographic patterns, identifies trends in terrorist activities, and presents key insights through visualizations and interactive dashboards.

## Project Highlights

* Analyzed **180,000+ terrorism incident records** spanning 1970–2017.
* Performed data cleaning and exploratory data analysis (EDA) using **Pandas** and **NumPy**.
* Identified significant temporal, regional, and country-level terrorism trends.
* Created data visualizations using **Matplotlib** and **Seaborn**.
* Developed an interactive **Power BI dashboard** for deeper exploration of the data.
* Derived insights from data originally compiled and maintained by **START, University of Maryland**.

## Running the Project

1. Clone the repository:

```bash
git clone https://github.com/PrakharPapnai/Global-Terrorism-Dataset-Analysis-.git
```

2. Download the dataset using the link above.

3. Place the CSV file in the project directory.

4. Open and run:

```text
Code.ipynb
```

5. To explore the dashboard, open:

```text
EDA_Dataset_Terrorism.pbix
```

using Microsoft Power BI Desktop.
